■ AMBIENT air quality monitor
not connected
→ point this at your ESP32's IP address
01 — Live Reading

Calls GET /api every 2 seconds and draws the latest PM2.5, CO₂, temperature and humidity values to the canvas. A good first sketch — fetch one endpoint, display the values.

02 — History Graph

Calls GET /api/history once (then refreshes every 10 s) and plots CO₂ and PM2.5 as line graphs over time. Demonstrates working with arrays of JSON objects.
